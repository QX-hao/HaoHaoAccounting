# Web Features

Vertical Web product modules.

## Modules

- `auth`: login page and auth requests.
- `overview`: dashboard/home overview.
- `transactions`: add transaction, AI parse, and transaction list.
- `reports`: report summary visualizations.
- `accounts`: account management.
- `categories`: category management.
- `dataio`: import/export.
- `settings`: current user/profile view.

Route files under `web/app/*` should import feature pages from here and stay thin.
